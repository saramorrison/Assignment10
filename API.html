The API selected is a quote of the day API which gives the user a new inspirational quote every time the webpage is refreshed.

The website that I found the API on is: https://favqs.com/api/qotd

This API could be helpful for a meditation application or a wellness store's website. Inspirational quotes would be commonly found on either of these sites and an API that provides a new quote each time the site is reloaded would be very useful.
